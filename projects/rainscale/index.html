RainScale-EDSR

2.5× spatial super-resolution of daily rainfall over Java Island — bridging coarse climate output to operational hydrology precision.

RainScale-EDSR applies super-resolution convolutional networks to the problem of spatial downscaling in climate science. The model takes coarse-resolution (0.25°) daily rainfall data from MSWEP V2 and upscales it to 0.1° — a 2.5× spatial leap that bridges global climate model output and the precision required for operational hydrology over Java Island.

The architecture is EDSR (Enhanced Deep Super-Resolution) with 8 residual blocks adapted for single-channel atmospheric data, plus a Conv2DTranspose 2× upsampling head. A companion ResNet-SR baseline was trained for comparison.

Training used 5,113 frames through a 5-stage preprocessing pipeline: log-ε transform, stratified RobustScaling by rain-intensity tier, outlier clipping, Sobel/Gaussian spatial feature extraction, and MinMax normalization. Evaluated on 1,096 held-out test frames.

Implementation Highlight

EDSR residual block + 2× Conv2DTranspose upsampling head 
def residual_block(x, filters=64):
    skip = x
    x = Conv2D(filters, 3, padding='same', activation='relu')(x)
    x = Conv2D(filters, 3, padding='same')(x)
    return Add()([x, skip])

def build_edsr(num_res_blocks=8, filters=64, scale=2):
    inputs = Input(shape=(None, None, 1))
    x = r = Conv2D(filters, 3, padding='same')(inputs)
    for _ in range(num_res_blocks):
        x = residual_block(x, filters)
    x = Add()([Conv2D(filters, 3, padding='same')(x), r])
    x = Conv2DTranspose(filters, 3, strides=scale, padding='same')(x)
    return Model(inputs, Conv2D(1, 3, padding='same')(x))
