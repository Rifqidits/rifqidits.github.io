RFID Parking System

Multi-card RFID access control with EEPROM persistence, Hall-effect occupancy sensing, and real-time slot display on Arduino Uno.

An Arduino-based car parking management system combining access control, occupancy sensing, and a real-time physical interface. The MFRC522 RFID reader (SPI) authenticates vehicles against authorized card UIDs stored in EEPROM — persisting across power cycles without reflashing. A master-card programming mode allows runtime addition and removal of cards.

The KY-024 Hall effect sensor (analog A0 + digital D2) detects slot occupancy via magnetic field change. An infrared sensor triggers automatic gate closure on vehicle exit. Gate actuation uses a relay-controlled solenoid with a 10-second access window, plus a servo motor for the physical barrier.

Feedback is provided by a red/green LED pair for occupancy indication and a 16×2 I2C LCD for live slot availability display. All components are coordinated in a single event loop, with a buzzer for audio feedback on access-granted and programming-mode events.

Implementation Highlight

EEPROM-backed UID authentication — runtime card registration without reflashing 
void checkAccess(byte *uid) {
  byte storedCount = EEPROM.read(0);
  for (byte i = 0; i < storedCount; i++) {
    bool match = true;
    for (byte j = 0; j < 4; j++) {
      if (uid[j] != EEPROM.read(1 + i * 4 + j)) {
        match = false;
        break;
      }
    }
    if (match) { grantAccess(); return; }
  }
  denyAccess();
}

void grantAccess() {
  digitalWrite(RELAY_PIN, HIGH);
  myServo.write(90);          // open gate
  lcd.print("Access Granted");
  tone(BUZZER_PIN, 1000, 200);
  delay(ACCESS_WINDOW_MS);
  digitalWrite(RELAY_PIN, LOW);
  myServo.write(0);           // close gate
}
