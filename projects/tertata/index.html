terTATa-LLM

Financial question-answering for Indonesia's 66M+ MSME operators — Top 10 nationally, Datathon 2025.

terTATa-LLM is a financial question-answering system built for Indonesia's 66 million+ micro, small, and medium enterprise (MSME) operators. It answers hybrid tabular-text queries over structured financial reports — a task that demands simultaneous understanding of natural language and numeric tables.

The base model is Nous-Hermes-2-Mistral-7B-DPO, fine-tuned via LoRA (r=64, α=16) with 4-bit NF4 quantization on 13,215 TAT-QA financial table-text question-answer pairs. Only 0.72% of the model's 3.78 billion parameters were updated during training, making the adaptation highly efficient without sacrificing domain performance.

The model uses a 6-step structured reasoning chain for hybrid tabular-text queries. The final adapter was deployed on Hugging Face Spaces and evaluated on a 1,669-sample held-out test set, achieving F1 = 46.62.

Implementation Highlight

LoRA configuration — only 0.72% of 3.78B parameters trained 
from peft import LoraConfig, get_peft_model

lora_config = LoraConfig(
    r=64,
    lora_alpha=16,
    target_modules=["q_proj", "v_proj"],
    lora_dropout=0.05,
    bias="none",
    task_type="CAUSAL_LM",
)

model = get_peft_model(base_model, lora_config)
model.print_trainable_parameters()
